# madeleine interpolation tool

Usage:

> ./madeleine <_mesh1path_>/<_mesh1.stl_> <_mesh2path_>/<_mesh2.stl_>

Interpolation done from _mesh1_ to _mesh2_.

Default mesh files: ./data/sphere1.stl  ./data/sphere2.stl
